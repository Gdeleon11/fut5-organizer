# Fut5 Organizer — sistema visual

Set de banners generado para la navegación principal. Todas las imágenes comparten
una dirección de fotografía deportiva nocturna, cancha de fútbol 5, fondo verde
oscuro y acentos `#39e55a`. La acción se concentra a la derecha para reservar el
lado izquierdo al título de la interfaz.

Prompt base: fotografía deportiva editorial y cinematográfica de fútbol 5 amateur
en Guatemala, de noche bajo reflectores; composición ultrapanorámica para banner,
acción a la derecha y espacio negativo oscuro a la izquierda; negro verdoso,
verde bosque y verde eléctrico; sin texto, marcas, logos ni uniformes profesionales.

El concepto específico de cada imagen corresponde literalmente a su sección:
inicio del partido, huddle de equipo, tarjeta de disciplina, pago compartido,
retrato de jugador, comunidad, organización con tablet, plantel, cancha vacía,
pizarra táctica, celebración de torneo y administración de varias canchas.
